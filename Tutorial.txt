- Setting nama owner:
• Cari teks "owner" di bagian file config.js untuk mengedit.
Contoh 👇🏻 
owner: 
    name: "YOUR NAME", // Nama owner
    number: ["628..."], // Format: 628xxx (tanpa + atau 0)

- Setting nama pembuat stiker dan author:
• Cari teks "sticker" di bagian file config.js untuk mengedit.
Contoh 👇🏻 
sticker: 
    packname: "𝗢𝗨𝗥𝗜𝗡 𝗔𝗜", // Nama pack sticker
    author: "YOUR NAME", // Author sticker

- Setting saluran channel WhatsApp, nama channel WhatsApp dan id saluran, id saluran bisa didapatkan dari bot WhatsApp lain jika belum siap jadibot:
• Cari teks "saluran" di bagian file config.js untuk mengedit.
Contoh 👇🏻 
saluran: {
    id: "Nomor id saluran@newsletter", // ID saluran (contoh: 120363xxx@newsletter)
    name: "CHANNEL NAME", // Nama saluran
    link: "LINK YOUR CHANNEL", // Link saluran

- Setting energy limit owner dan semua user:
• Cari teks "energi" di bagian file config.js untuk mengedit.
Contoh 👇🏻 
energi: {
    enabled: true, // Jika true, maka sistem energi/limit akan bekerja
    default: 4500, // Ini limit default dari semua user kecuali owner dan premium, sesuaikan sendiri jika kurang tepat 
    premium: 99999999, // Sesuaikan sendiri jika kurang tepat 
    owner: -1, // Jangan di edit ke angka 1+, gunanya sudah disetting dari sana angka minus 1- sama dengan unlimited energy, khusus owner saja

- Jangan lupa private bot agar tidak menggangu penggunaan lain, daftar mode untuk private bot: 
- .self untuk mengecualikan semua orang untuk menggunakan command 
- .public memungkinkan semua orang bisa mengakses kembali bot yang sudah di private 
- .publicgc fungsinya untuk membuka mode private... khusus grup yang ingin di public saja, fitur ini akan bekerja jika command .self diaktifkan 
- .onlygc mengecualikan akses command di private chat dan di alihkan ke grup untuk mengakses command
